## 安裝

建議 node.js 為 v22.13.1 版

### 取得專案

```
git clone https://github.com/169628/PRAC_tailwind_hotel.git
```

### 移動到專案內

```
cd PRAC_tailwind_hotel
```

### 安裝套件

```
npm install
```

### 運行專案

```
npm run dev
```

在以下網址可以看到畫面

```
http://localhost:5173/
```
